# SimplifyPro — Index projet

## En bref

Site web en anglais pour promouvoir et vendre l'acces a un indicateur
de trading TradingView (SimplifyPro V6.0) par abonnement.

## Stack technique

- Next.js 15.4.x + Payload CMS 3.x (localisation native)
- Stripe (Setup Intent, Customer Portal, Coupons, Webhooks)
- shadcn/ui + Tailwind CSS (theme fintech sombre)
- TypeScript strict
- Resend (emails transactionnels)
- PostHog (analytics)
- Vercel Pro + PostgreSQL (Neon/Supabase)

## Modele economique

- Essai gratuit : 7 jours (CB requise, demarre a l'activation TradingView)
- Mensuel : 49$/mois
- Annuel : 349$/an (~29$/mois, 40% reduction)
- Promos : ~24,90$/mois (fetes, evenements, via Stripe Coupons)

## Cible

Traders intermediaires (25-40 ans), Indices/Forex/Crypto, utilisateurs TradingView.
